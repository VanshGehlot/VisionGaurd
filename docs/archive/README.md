# Archive

This folder keeps non-runtime artifacts out of the repository root while preserving useful submission history.

## Folders

- `design-iterations/` contains screenshots and visual proof from earlier UI iterations.
- `legacy-static-ui/` contains the pre-React static frontend files retained for reference only.

The active product frontend is `frontend/`. The active backend entrypoint is `api.py`. Gradio fallback remains `app.py`.

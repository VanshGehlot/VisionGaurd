# Recording Checklist

Before recording:

- [ ] AMD endpoint validated
- [ ] `python main.py` works
- [ ] `python app.py` works
- [ ] Good image returns PASS
- [ ] Defective image returns STOP_LINE or ALERT_OPERATOR
- [ ] Event log populated
- [ ] Shift report generated
- [ ] README open in another tab
- [ ] Architecture diagram ready
- [ ] Terminal with ROCm/vLLM proof ready
- [ ] Browser zoom at 90–100%
- [ ] Cursor movement slow
- [ ] No file picker fumbling
